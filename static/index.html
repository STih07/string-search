Параметр Назначение Тип Пример
exact Искать только прямое совпадение boolean (по умолчанию true) example.com/search?exact=false
NAME Поиск по полю NAME string example.com/search?NAME=Валентин
EDRPOU Поиск по полю EDRPOU string example.com/search?EDRPOU=33319045
ADDRESS Поиск по полю ADDRESS string example.com/search?ADDRESS=20100&exact=false
